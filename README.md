# 🚀 Jornada QA - Qualidade de Software

Repositório criado para consolidar meus estudos práticos em Engenharia de Qualidade de Software, focado em metodologias ágeis, testes de caixa preta, APIs e automação.

## 📁 Estrutura do Repositório

* `/cenarios_login_anhanguera.md`: Planejamento e escrita de cenários de teste manuais (Fluxo Feliz e Negativos) baseados na tela de login da faculdade Anhanguera.

* `/serverest_api_tests.json`: Coleção de testes automatizados de API (Cadastro e Login) desenvolvida no Postman utilizando scripts em javaScript para validação de Status Codes (200, 201 e 401).
